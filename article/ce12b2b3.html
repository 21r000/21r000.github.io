# 下载破解项目

使用 GitHub 上的 typoraCracker 项目进行破解

使用 git 克隆脚本文件（需配置好 git）或者直接下载即可

# 安装依赖项

通过 pip 命令对 requirements.txt 文件中的依赖项进行安装

找到 Typora 安装目录下的 resources 目录

找到 resources 目录下的 app.asar 文件

默认目录是在 C:\Program Files\Typora\resources\app.asar ，如有更改请自行更换。

# 授权码生成

需要安装 nodejs 环境，如果没有安装请自行安装。

# 激活

授权码输入生成的码，邮箱输入 crack@example.com ，完事。

# 弊端

Typroa 在重新安装之后，可能会出现 pdf 导出卡顿等问题

原因

因为 Typroa 在删除的同时也关闭了 Print Spooler 服务

# 解决方法

在键盘上按 window + r ，输入 services.msc ，回车

找到 Print Spooler 服务，并开启该服务

设置开机自启动

在键盘上按 window + r ，输入 regedit ，回车

找到以下键值： HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Spooler

选定 Spooler 这个文件夹，在右边窗口里找到 DependOnService 这个键值。

如果存在 http 将其删除，只留 RPCSS ，确定后退出注册表编辑器，重启计算机即可！

PS：破解后部分功能可能会出现失效情况！

建议使用正版！！！
