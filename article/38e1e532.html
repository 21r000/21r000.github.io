# 破解记录

双击安装程序安装 Burp Suite Professional 2022.2.2 ，一路默认即可

打开 Burp Suite Professional 2022.2.2 安装路径

将 BurpLoaderKeygen.jar 放在 Burpsuite 安装路径下

双击运行 BurpLoaderKeygen.jar ，打开 ok

直接点 run 运行

提示输入 License key 了

->Next

->Manual activation

将 request 放入注册机的 Activatiion Request 中

将自动生成 Response 放入 Burpsuite 中

->Next ，激活成功

后面一路 Next 就大功告成了
