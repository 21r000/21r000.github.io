# 虚函数

虚函数是 C 中的一个概念，而众所周知 C 是一门面向对象的语言。

C++ 的特性：

  • 封装

  • 集成

  • 多态

虚函数是在程序运行时定义的函数，它的地址不能在编译的时候决定，只有在调用的时候才能确定虚函数的地址，所有对虚函数的引用通常都放在一个专属的数组中，也就是虚函数表 Virtual Table 中，缩写为：VTBL，也就是：Virtual Table List。数组的每一个元素都存放在这个类中虚函数的地址，在调用虚函数时，先取出虚函数表指针，也就是指向 VTBL 的指针，得到虚函数表的地址就是首地址，再根据这个地址到虚函数表中取出函数的地址，取出地址之后再用汇编语言中的 call 来进行调用。
