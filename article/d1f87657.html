VX-WeChatWin.dll文件修改实现防撤回与多开

# 实验环境

  • x32dbg / x64dbg
  • Windows 10 专业版 / Windows 7 专业版
  • 微信 Windows 版 3.6.0.18

# 信息摘要

  • 撤回、限制多开的逻辑都位于 WeChatWin.dll 文件中
  • 撤回相关的关键词 revokemsg
  • 多开相关的关键词 WeChat_App_Instance_Identity_Mutex_Name
  • WeChatWin.dll 文件默认路径为： C:\Program Files (x86)\Tencent\WeChat\[3.6.0.18]

# 调试修改

打开 dbg，在左上角选择文件 -> 附加或者 Alt + A ，打开微信进程

切换到 符号 选项卡，在左下角搜索 WeChatWin.dll ，双击 wechatwin.dll 进入 CPU 选项卡

右键 搜索 -> 当前区域 -> 字符串

# 防撤回调试

直接搜索关键词 revokemsg ，然后依据关键函数 mov ecx,wechatwin.xxxxxxxx 双击 "revokemsg" 进入

需要进行修改的是当前行的上面一行： je wechatwin.xxxxxxxx

对 je wechatwin.xxxxxxxx 进行汇编，将 je 修改为 jmp 即可

# 多开调试

直接搜索关键词 WeChat_App_Instance_Identity_Mutex_Name ，然后双击第一个 L"WeChat_App_Instance_Identity_Mutex_Name" 进入

需要进行修改的是当前行的上面第一个出现的 push ebp

注：这里容易错误修改了 push ebx ，也即是我们的启动函数，从而可能导致 VX 无法正常启动

对 push ebp 进行汇编 ，将 push ebp 修改为 ret 即可

# 生成补丁

点击生成补丁的按钮，然后点击修补文件就可以得到修改后的 WeChatWin.dll
