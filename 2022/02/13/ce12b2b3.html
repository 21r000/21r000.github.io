# Typroa 安装

进入官网下载安装所需的 Typroa

官网链接

双击进行安装

一路默认安装即可，安装地址如有需求可自行更改

安装完成自启也可取消

可以看到此时的 Typroa 处于未激活状态

# 项目破解

# 下载破解项目

使用 GitHub 上的 typoraCracker 项目进行破解

使用 git 克隆脚本文件（需配置好 git）或者直接下载即可

# 安装依赖项

通过 pip 命令对 requirements.txt 文件中的依赖项进行安装

找到 Typora 安装目录下的 resources 目录

找到 resources 目录下的 app.asar 文件

默认目录是在 C:\Program Files\Typora\resources\app.asar ，如有更改请自行更换。

# 执行解包命名

Windows 下如果路径是有空格的，需要用 "" 包起来，如果提示权限问题，用管理员身份运行命令行，建议后续都用管理员身份执行。

使用脚本文件夹 example\path\License.js

替换掉 workstation\outfile\dec_app\License.js 文件

打包 app.asar 文件

如遇报错:

手动在脚本路径下打包 workstation/outappasar 文件夹就行了:

将打包回来的 app.asar 文件重新丢到 Typora 的 resources 目录下

如恐意外可将原文件复制一份

# 授权码生成

需要安装 nodejs 环境，如果没有安装请自行安装。
